`manuscript.tex` is the LaTeX version of the current CKD paper draft.

It expects analysis outputs in `../outputs/`:

- CSV exports for tables and PGFPlots figures in `../outputs/latex_data/`
- Legacy PNG figures may still be generated by the analysis script, but the paper
  render uses LaTeX-native figures.

Compile command:

```bash
cd paper
/Users/jxchu/bin/tectonic manuscript.tex --outdir build
```
